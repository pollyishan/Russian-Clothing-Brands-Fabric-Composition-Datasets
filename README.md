# Russian Clothing Brands Fabric Composition Datasets

Датасеты по составу тканей российских брендов одежды.

## Формат
CSV

## Источник
Данные собраны скриптом на Selenium или с помощью парсеров-расширений для браузера из открытых карточек товаров брендов.

## Ограничения
Информация может меняться на сайтах брендов.
Товарные знаки, названия брендов и исходный контент принадлежат их владельцам.
